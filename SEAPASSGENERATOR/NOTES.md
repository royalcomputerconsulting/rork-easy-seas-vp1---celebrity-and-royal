# Notes

## Scope of this backup pack
This folder is focused on the SeaPass Generator feature and its direct integration points.

It includes:
- the full current generator screen
- the SVG/rendering engine
- native and web export logic
- admin gate dependency snapshot
- backend proxy snapshot
- theme and error boundary support files
- route/settings integration snippets
- package and tsconfig snapshots
- environment and behavior notes

## What is intentionally captured as snapshots instead of live copies
Some project-wide files are extremely large or unrelated to the feature. For those, this pack stores either:
- direct snippets, or
- current snapshots relevant to the feature

Examples:
- settings integration is stored as a snippet
- root stack integration is stored as a snippet
- package files are stored as `*.snapshot.json`
