## Chrome Extension Setup

The Chrome extension automates data collection from the Royal Caribbean website.

### Installation Steps

1. **Download the Extension**
   - Go to Settings → Data Management → Browser Extension
   - Click "Download Chrome Extension"
   - Save the ZIP file to your computer
   - Extract the ZIP file to a folder

2. **Install in Chrome**
   - Open Chrome browser
   - Navigate to `chrome://extensions`
   - Enable "Developer mode" (toggle in top right)
   - Click "Load unpacked"
   - Select the extracted extension folder
   - Extension icon appears in toolbar

3. **Using the Extension**
   - Sign in to Royal Caribbean Club Royale website
   - Go to your offers page
   - Click "SHOW ALL OFFERS" button (added by extension)
   - Wait for all offers to load
   - Click "EXPORT CSV" button
   - Note the number of rows in the exported file
   - Click "SCRAPE WEBSITE" button
   - Wait 20-30 minutes for scraping to complete
   - OFFERS.CSV file downloads automatically

4. **Import the Data**
   - Open EasySeas app
   - Go to Settings → Data Management → Import
   - Click "Offers CSV"
   - Select the downloaded OFFERS.CSV file
   - Wait for import confirmation
   - Data appears in Overview tab

### Troubleshooting Extension
- **Extension not loading:** Check Developer mode is enabled
- **Buttons not appearing:** Refresh the Royal Caribbean page
- **Scraping fails:** Check internet connection, try again
- **Import fails:** Verify file format, check error message

## Troubleshooting

### Common Issues

#### Data Not Showing
**Problem:** Imported data doesn't appear in app
**Solution:**
1. Check import success message
2. Refresh the tab by switching and returning
3. Force close and reopen app
4. Check Settings → Data Overview for counts
5. Try "Restore from Backup" with exported file

#### Extension Not Working
**Problem:** Chrome extension buttons don't appear
**Solution:**
1. Refresh Royal Caribbean page
2. Verify extension is enabled
3. Check you're on the correct offers page
4. Disable other extensions temporarily
5. Try incognito mode

#### Points Not Calculating
**Problem:** Tier progress not updating
**Solution:**
1. Manually update points in Settings → User Profile
2. Save profile changes
3. Check sessions are logged correctly
4. Verify coin-in amounts in sessions
5. Recalculate in Analytics tab

#### App Running Slow
**Problem:** App performance degraded
**Solution:**
1. Clear expired data
2. Archive old cruises
3. Reduce number of open modals
4. Restart the app
5. Clear app cache if available

#### Authentication Issues
**Problem:** Can't log in or stay logged in
**Solution:**
1. Verify email is whitelisted (contact admin)
2. Check internet connection
3. Try different browser (if web version)
4. Clear browser cache
5. Request whitelist verification
